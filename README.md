### 📂 Repository Structure

This website is built with a simple static site generator and organized as follows:

- `_data/` : Configuration files, like the site's navigation menu.
- `_includes/` : Reusable HTML components (e.g., page head, author profile).
- `_pages/` : The main content pages (About, Publications, Projects, etc.).
- `assets/` : Static files (CSS stylesheets, images, downloadable PDFs).
- `_config.yml` : Main configuration file for the site.

### 🛠️ Local Development

To run this site locally:

1.  **Clone the repository**
    ```bash
    git clone https://github.com/ZhongyaoTuo/ZhongyaoTuo.github.io.git
    cd ZhongyaoTuo.github.io
    ```

2.  **Install dependencies**
    ```bash
    bundle install
    ```

3.  **Run Jekyll server**
    ```bash
    bundle exec jekyll serve
    ```

Your site will be available at `http://localhost:4000`.
